ant design 在项目中应用的一些不同

Button   type属性为danger 时，除非手动点击才会出现红色背景，与组件有差别

        项目中是直接显示红色背景(现需手动更改)

ICON   一般在项目菜单栏用到，其他地方几乎用不到

Grid 栅格 布局

    使用Row和Col进行布局(类似于Flex布局)

Layout布局

    属于整体的布局 :

        包含Header Sider  Content Footer

Affix 固钉

    内容区域较长，页面滚动时，如导航需要再滚动范围内始终展现

    常用于侧边菜单和按钮

    缺点 : 会遮挡页面内容

Bresdcrumb面包屑

    用于页面跳转时导航的层级结构

    可带图表显示，注 : 图表在文字前面(一般是不带图表显示的)

    可跳转当前所展示的导航结构页面

    简单来说就是tab的切换页

Dropdown 下拉菜单

    类似于select的下拉菜单，不同的是此样式非常简洁，如在项目中应用需自行添加样式

    功能 : 

        移入菜单自动出来，或点击时菜单会展开

Menu 导航菜单

    属于通用布局

    常用于左边菜单栏的展示(全局公用)

    内部可内嵌菜单，内嵌菜单可以被收缩和展开

Pagination 分页

    列表页数据较多，使用分页进行数据展示

    注 : 只接收后台返回数据为list的数组，否则页面无法展示

    此组件是结合列表返回的数据的一个Pagination对象进行展示可分为多少页

    缺点 :

        会默认展示跳转到多少页的输入框，但是不会骑作用，需要手动在封装的公共的index.js文件夹里面添加以下代码，才会生效

            const paginationProps = {
                showSizeChanger: true,
                showQuickJumper: {
                    goButton: <Button className="ant-pagination-go-button">跳转</Button>
                },
                ...pagination
            }

PageHeader 页头

    主要用于声明页面猪蹄，展示用户当前页面的信息(通常称为导航，带面包屑的导航)

Steps 步骤条

    当任务复杂或者存在先后关系时，可分解成一系列步骤(清晰)

    可添加多步，方向可自由调试

    主要由current属性进行控制当前的步骤

AutoComplete 自动完成

    功能 : 

        输入框自动完成(在输入框中输入，会自动下拉出现所输入的相关内容)

        模糊搜索

CheckBox 多选框

    可以进行多项选择

Cascader 级联选择

    如省市区、公司层级、事物分类

    特点，类似select组件，但辞组件可以在一个浮层完成选择

DatePicker 日期选择框

    用户需要输入日期，可以点击标准框，弹出日期进行选择

Form 表单

    具有数据收集、校验、提交的功能，包含复选框、单选框、输入框、下拉选择框等元素

    排列方式 : 水平排列、垂直排列、行内排列


InputNumber 数字输入框

    限制值输入数字也可通过step来限制小数点的精度

    特点 : 鼠标移入有上下箭头，可增可减

Input 输入框

    在项目中用于模糊搜索，或输入框的美容输入和展示

Mentions 提及

    用于在输入中提及某人，常用于发布
    聊天或评论功能

Rate 评分

    主要是用图标星星进行评价展示

Radio 单选框

    在多个选择中做单个选中状态

Switch 开关

    功能 : 开关选择器

Slider 滑动输入条

    用于展示当前值和可选范围

    展示有点类似于进度条


Select 选择器

    下拉选择器

    功能 : 下拉菜单给用户操作选择

TreeSelect 数选择

    类似Select的组件，但是下拉出来的是一个树形结构

    用于层级展示、公司层级、分类目录等

Transfer 穿梭框

    功能 :

        如有两个选择框 A 、B

        在A中选择的内容可直接进到B中反之，B选择的内容也可以直接进到A中

TimePicker 时间选择框

    功能 : 

        是属于点击选择时间，下拉进行滑动选择时间内容

        选择框可设置大、中、小(默认中)

Upload 上传

    功能 : 
    
        文件上传、拖拽(可设置上传文件的大小、可删除、可设置上传个数)

        上传图片(格式限制、大小限制)，可直接进行展示多个图片，鼠标滑入出现删除和放大的图表，可进行操作

        缺点 : 如果只想进行删除和放大其中的一个功能操作，只能通过css样式将其隐藏

Avatar 头像

    用来用户或事物，支持图片、图标或字符展示

    头像尺寸可进行大小调试、两种形状可选(圆形、正方形)

    可自定义图标颜色及背景色

    对于头像字符较长时，字体大小可更具头像宽度自动调整

Badge 徽标数

    用于图标右上角的圆形徽标数字

Comment  评论

    对网址内容的反馈、评价和讨论(一般用于消息通讯等)

Collapse 折叠面板

    功能 :

        可折叠/展开的内容区域，进行分组展示(允许单个内容展开/隐藏)


Carousel 走马灯

    类似于轮播图(可选择对个类型的切换)

Card 卡片

    样式是卡片容器

    可承载文字、列表、图片、段落(常用于后台概览页面) 卡片的边框背景可自由调试

Calendar 日历

    按照日历的形式展示数据

    当数据是日期或按照日期划分时，如:日程、课表、价格、日历等(目前支持的是年/月切换)

    容器可嵌套到有控件的容器中

    头部内容可自定义


Descriptions 描述列表

    功能 :

        成组展示(用于详情页的信息展示)

        边框、背景、大小可自由调试

Empty 空状态

    用于全局的无数据情况下的一个展示(下拉、穿梭框、列表等)

List 列表

    可设为通用列表(常用于后台数据展示)

    单独使用可下拉加载更多

Popover  气泡卡片

    点击/鼠标移入，弹出气泡式的卡片浮层

    气泡可承载内容，刻在里面进行其他操作

Statistic 统计数值

    展示统计数值

    可设置小数点、千分位标识符、数值精度、数值的后缀、数值标题、数值内容、数值样式

    可进行待机室展示

Tree 属性控件

    功能 : 

        文件夹、分类、层级关系

        可使用多选、禁用、选中、默认展开、将节点拖拽到其他节点内部或前后、动态数据加载、模糊搜索、连接线展示、不同节点制定图标

Tooltip 文字提示

    只支持图示气泡框

Timeline 时间轴

    垂直展示的时间流信息

Tag 标签

    用于标记和分类的小标签

    也可用于关闭标签、给标签增加添加或删除动画

Tags 标签页

    功能 : 选项卡切换

Table 表格

    展示行列数据

    可对数据进行排序、搜索、分页等

Alert 警告提示

    展示需要关注的信息，通过不同的图标和颜色进行区分

Drawer 抽屉

    屏幕边缘滑出的浮层面板

Modal 对话框

    进行点击操作，弹出弹框，里面可进行其他功能操作，可设置标题

Message 全局提示

    全局展示操作反馈信息

    如 : 点击保存提示用户保存成功

    顶部居中显示自动消失，不会打断用户操作

Notification 通知提醒框

    全局展示通知提醒信息(在系统的四个角通知),一般都是需要手动关闭弹框

    用于消息推送，交互通知

Progress 进度条

    展示当前进度和状态

    以标准的进度条展示、以圆圈的形式展示、进度条可为渐变色

Propconfirm 气泡确认框

    点击元素弹出气泡式的确认框

Result 结果

    用于反馈一系列操作任务的处理结果(如:成功、失败、没有权限、页面找不到 等)

Spin 加载中

    用于页面和区块的加载状态

Skeleton 骨架屏

    在需要等待加载内容的位置提供一个占位图形

    使用场景 :

        网络较慢、图文信息内容较多、适合第一次加载数据

Anchor 锚点

    用于跳转到页面指定位置

BackTop 回到顶部

    返回到顶部的操作按钮

ConfigProvider  全局化配置

    语言、分页、下拉、日期、弹出框、空白数据

Divider 分割线

    内容的分割线(是否虚线、位置、水平或垂直)
